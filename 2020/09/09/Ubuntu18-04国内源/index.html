前期准备

如果是ubuntu18.04以上的图像化界面大不可如此复杂，可直接去软件与更新图像化操作选择国内源
（建议）复制原来的源文件：
sudo mv /etc/apt/sources.list /etc/apt/sources.list.bak
然后新建sources.list文件，挑选快速的国内源粘贴进去
sudo vim /etc/apt/sources.list
然后刷新一下源就好了
sudo apt update
