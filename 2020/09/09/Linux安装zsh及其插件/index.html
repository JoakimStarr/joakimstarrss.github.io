Created2020-09-09|Updated2020-09-09|Linux
|Post View:

前言

Linux有许多的发行版本，安装软件都大同小异，这里我便以Ubuntu实例

安装zsh

首先安装zsh
sudo apt install zsh

安装oh-my-zsh及其插件

安装oh-my-zsh:

安装命令高亮插件:

在.zshrc文件中的plugins一栏加入zsh-syntax-highlighting这一句
vim .zshrc
加入zsh-syntax-highlighting
plugins=( git zsh-syntax-highlighting )
立即刷新命令 source .zshrc
安装命令历史插件:

在.zshrc文件中plugins一栏加入zsh-autosuggestions
plugins=( git zsh-syntax-highlighting zsh-autosuggestions)
立即刷新命令 source .zshrc
安装autojump自动跳转插件
sudo apt install autojump
然后在.zshrc的底部加上
. /usr/share/autojump/autojump.sh
立即刷新命令 source .zshrc
注意：前面有一个点，点后面有一个空格，万不可忽略
使用方法：
比如你安装成功后进入了Download文件夹
你可以使用 j d命令便可进入Download文件夹，wj 文件夹名（支持模糊查询） 可直接逃过中间的一些文件夹，直接跳到你经常使用的文件夹里去，
